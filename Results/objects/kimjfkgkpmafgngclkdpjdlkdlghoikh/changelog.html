Extension Changelog

Formatting Information

Version Numbers

Version numbers are signified by [ X.Y.Z ], as this project adheres to Semantic Versioning

X version for major updates

Y version for minor updates

Z version for patch updates

Version Dates

All dates are written in the International Standard ISO 8601 notation format of YYYY-MM-DD

Version Tags

[ Added ] for new features

[ Changed ] for changes in existing functionality

[ Deprecated ] for once-stable features removed

[ Fixed ] for any bug fixes

Unreleased
